  • 开启了 flex 布局的元素叫 flex container(容器)
  • flex container 里面的直接子元素叫 flex items(项目)

应用在 flex-container 上的 css 属性

  • flex-direction：决定主轴(main axis)方向，默认沿着主轴从 main start 开始往 main end 方向排布–从左往右

  • justify-content：决定了 flex items 在主轴上的对齐方式

  • align-items：决定了 flex items 在交叉轴(cross axis)上的对齐方式(默认从上到下)

  • flex-wrap：决定了 flex container 是单行还是多行

  • flex-flow：是 flex-direction || flex-wrap 的简写

  • align-content：决定了多行 flex items 在交叉轴(cross axis)上的对齐方式，用法与 justify-content 类似

应用在 flex-items 上的 css 属性

  • order：决定 flex items 的排布顺序

    • 可以设置任意整数，值越小排在越前面
    • 默认值为 0

  • align-self：可通过 align-self 覆盖 flex container 设置的 align-items

    • auto，默认值，遵从 flex container 的 align items 设置
    • stretch、flex-start、flex-end、center、baseline，效果与 align-items 一致

  • flex-grow：当 flex container 的 size 有剩余时，items 分剩余 size

    • 可以设置任意非负数字，默认值是 0
    • 当 flex container 在 main axis 方向上有剩余 size 时，此属性才会生效
    • 若 flex grow 总和超过 1，每个 flex items 扩展的 size 为：flex container 的剩余 size * flex-grow / sum
    • 若综合不超过 1，每个 flex items 扩展的 size 为：flex container 的剩余 size * flex-grow 的值

  • flex-shrink：收缩，默认值为 1

  • flex-basis：用来设置 flex items 在主轴上的元素大小

    • auto：默认值；具体的宽度数值
    • 优先级
      • max-width/height min-width/height
      • flex-basis
      • width height
      • 内容本身的 size

  • flex：flex-grow flex-shrink flex-basis 的简写，flex 属性可以指定 1、2 或 3 个值

css 书写顺序

  1. 位置属性：position、top、right、z-index、display、float 等
  2. 大小：width、height、padding、margin 等
  3. 文字系列：font、line-height、letter-spacing、color、text-align 等
  4. 背景：background、border 等
  5. 其他：animation、transition 等

连字符 CSS 选择器命名规范

  1. 长名称或词组可以使用中横线-来为选择器命名。
  2. 不建议使用_下划线来命名 CSS 选择器 ，为什么呢?
  • 输入的时候少按一个 shift 键;
  • 浏览器兼容问题(此如使用_ tips 的选择器命名，在 IE6 是无效的)
  • 能良好区分 JavaScript 变量命名(JS 变量命名是用_ )

css 视差滚动

background

  • background-color //背景颜色
  • background-position //背景图像的位置
  • background-size //背景图像尺寸
  • background-repeat //如何重复背景图像
  • background-origin //背景图片的定位区域
  • background-clip //背景的绘制区域
  • background-attachment //背景图像是否固定或者随着页面的其余部分滚动
  • background-image //要使用的背景图像
  • inhert //应该从父元素继承 background 属性的设置

user-select:用户能否选中文本

  • none : 元素及其子元素的文本不可选中

  • text : 文本可以被选中

  • auto : 文本将根据浏览器的默认属性进行选择

    • 在 ::before 和 ::after 伪元素上，采用的属性值是 none
    • 如果元素是可编辑元素，则采用的属性值是 contain
    • 否则，如果此元素的父元素的 user-select 采用的属性值为 all，则该元素采用的属性值也为 all
    • 否则，如果此元素的父元素的 user-select 采用的属性值为 none，则该元素采用的属性值也为 none
    • 否则，采用的属性值为 text

  • all : 在一个 HTML 编辑器中，当双击子元素或者上下文时，那么包含该子元素的最顶层元素也会被选中

  • contain：允许在元素内选择；但是，选区将被限制在该元素的边界之内

  • element（IE 专有别名）

    与 contain 相同，但仅在 Internet Explorer 中受支持。
